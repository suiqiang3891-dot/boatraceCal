## 当前决策

- 系统是决策支持工具，第一阶段不自动下注。
- 优化目标以长期回收率和风险调整后收益为主，命中率为辅助指标。
- 首版聚焦三连单，同时保留扩展其他票种的数据结构。
- 本项目独立负责数据导入契约、数据治理、建模、回测和推荐；外部数据只作为可替换输入，不依赖任何其他项目的目录、代码或运行状态。
- 每晚生成次日预案，比赛日按 T-30、T-15、T-10、T-5 更新，T-10 形成最终决策。
- 所有训练与回测按时间切分，并强制检查数据泄漏。
