  由日期、场地和比赛号确定。

  数量与唯一性，并规范化无序组合。

- 推荐决策：校验阶段、决策、置信度、概率、赔率、期望值、时点、单位数和原因码；
  预案不能伪装为最终选择，最终选择必须带赔率、期望值和原因。
- 作业契约：作业键可审计且确定；快照目标和生命周期状态固定；仅允许显式状态迁移。
- 错误分类：稳定错误码集合覆盖来源、调度、解析、数据质量、模型、策略与版本冲突。
